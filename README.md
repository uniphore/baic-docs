# baic-docs

Source for the **Uniphore BAIC** customer-facing installation guide, served via GitHub Pages.

- Live site: https://uniphore.github.io/baic-docs/
- Content: `index.md` (rendered with Just-the-Docs Jekyll theme via `_config.yml`)
- Master copy: this repo. The `docs/index.md` copy in `uniphore/x-forge-installer` is a mirror — edit here first.

## Updating

Edit `index.md`, push to `main`, Pages auto-publishes within ~1 minute.
